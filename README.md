# Myntra_Clone

A fully responsive **Myntra-inspired e-commerce website** that allows users to browse products, view details, and manage their shopping cart efficiently.

---

## 🚀 Features

* 🏷️ Browse products with images, price, and details
* 🛒 Add to cart functionality
* ❌ Remove items from cart
* 💰 Dynamic price calculation
* 📱 Fully responsive design (Mobile + Desktop)
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express (if used)
* **Database:** MongoDB (if used)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Yashveersinghthakur/Myntra-Clone.git
```

2. Navigate to project folder:

```
cd Myntra-Clone
```

3. Open in browser:

```
index.html
```
